# Система виявлення мін

## Огляд проєкту

Ця система призначена для виявлення та класифікації протипіхотних і протитанкових мін за допомогою різних типів сенсорів. Система використовує методи злиття даних (sensor fusion) для підвищення точності виявлення та зменшення кількості хибних спрацьовувань.

## Архітектура системи

Система побудована за принципами гексагональної архітектури (або портів та адаптерів), що дозволяє легко замінювати окремі компоненти і тестувати їх незалежно. Основні компоненти включають:

### Доменна модель (Domain Layer)
- `Device` - представляє фізичний пристрій з сенсорами для виявлення мін
- `Mission` - представляє операцію з розмінування
- `Scan` - представляє окремий сеанс сканування території
- `SensorData` - дані, отримані з різних сенсорів
- `DetectedObject` - потенційна міна, виявлена системою

### Сервіси додатку (Application Layer)
- `DeviceService` - керує життєвим циклом пристроїв
- `SensorFusionService` - обробляє та об'єднує дані з різних сенсорів для виявлення мін
- Інші сервіси для керування місіями, адміністрування системи тощо

### Інтерфейси (Ports)
- Репозиторії для зберігання даних
- API для комунікації з клієнтами
- WebSocket для отримання даних з пристроїв у реальному часі

### Інфраструктура (Infrastructure Layer)
- Адаптери для баз даних
- Реалізації веб-серверів
- Інтеграція з іншими системами

## Типи сенсорів

Система підтримує наступні типи сенсорів:
1. **ЛІДАР** - використовується для сканування поверхні та виявлення геометричних аномалій
2. **Магнітометр** - виявляє металеві частини мін
3. **Акустичний сенсор** - використовує звукові хвилі для виявлення підповерхневих об'єктів

## Алгоритми злиття даних

Система використовує наступні методи для обробки та злиття даних:
- Калманівська фільтрація для покращення точності вимірювань
- Байєсівська мережа для класифікації об'єктів
- Метод Демпстера-Шефера для об'єднання ймовірностей від різних сенсорів

## Процес виявлення мін

1. Пристрої виконують сканування території та передають дані через WebSocket
2. Система обробляє дані з кожного типу сенсорів
3. Алгоритм злиття даних об'єднує інформацію з різних сенсорів
4. Система класифікує виявлені об'єкти та оцінює їх небезпечність
5. Результати відображаються на карті для операторів
